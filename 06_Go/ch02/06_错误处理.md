
## 错误处理

- error 接口
  Go 语言引入了一个关于错误处理的标准模式，即 error 接口，该接口的定义如下:
  ```go
    type error interface {
        Error() string
    }
  ```
  对于大多数函数，如果要返回错误，大致上都可以定义为如下模式，将 error 作为多种返回值中的最后
  一个，但这并非强制要求:
  ```go
    func Foo(param int)(n int, err Error) {
        // ...
    }
  ```
  调用时的代码建议按如下方式处理错误情况:
  ```go
    n, err := Foo(0)
    
    if err != nil {
        // 错误处理
    } else {
        // 使用返回值n
    }
  ```
