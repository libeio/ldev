
常用的文本元素如下

|   文本元素  | 功能 |
|:-----------|:-----|
| `<b></b>`  | 加粗 |
| `<br>`     | 强制换行 |
| `<i></i>`  | 倾斜  |
| `<u></u>`  | 下画线 |
| `<del></del>` | 删除 |
| `<strong></strong>` | 加粗强调 |
| `<em></em>` | 倾斜强调 |
| `<ins></ins>` | 下划线强调 |
| `<s></s>`  | 删除强调 |
| `<small></small>` | 小号字体 |
| `<sub></sub>` | 添加下标 |
| `<sup></sup>` | 添加上标 |
| `<code></code>` | 计算机代码 |
| `<q></q>`   | 引用内容，实际上就是给内容加上引号 |
| `<abbr></abbr>` | 文本缩写 |
| `<mark></mark>` | 突出文本 |
| `<time></time>` | 日期时间 |
