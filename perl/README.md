### 为什么要使用 perl

支持浮点运算

支持 shell

够看的数据类型

比编译型语言使用方便

虽是脚本，但速度并不慢

很多函数的名称和使用上对接 C 语言

强大的正则表达式

支持进程管理

足够丰富的模块支持，如数据库、进程间通信等

也可以自定义模块

面向对象编程
