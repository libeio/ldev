
```html
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Hello JS</title>
        <script Language="JavaScript">
            var x;
            x = prompt("Input x please:", "10");
            if (x > 0) {
                document.write("x=", x, " is positive!");
            } else if (x < 0) {
                document.write("x=", x, " is negative!");
            } else {
                document.write("x=", x, " is zero!")
            }
        </script>
    </head>
    <body>
        
    </body>
    </html>
```

JS 脚本语言定义在 `<head></head>` 当中，开始标记是 `<script Language="JavaScript">`，结束标记是 `</script>`。
